Student Management System
Overview
This is a full-stack Student Management System developed using Flask (Python) and SQLite.
It provides a modern admin dashboard and student portal to manage academic data efficiently.
This project was developed as part of my 1st Year – 2nd Semester studies at university, under the modules:
  •	IWT (Internet & Web Technologies)
  •	DBMS (Database Management Systems)
________________________________________
Features
Admin Panel
  •	Manage students (Add, Edit, Delete)
  •	Manage courses, modules, and subjects
  •	Upload learning materials
  •	Assign and manage student grades
  •	Mark and track attendance
  •	Post announcements (general & course-specific)
  •	Reset student passwords
  •	Export student data (CSV)
  •	View analytics dashboard (charts)
Student Panel
  •	View personal profile
  •	Access course materials
  •	Check grades
  •	View attendance percentage
  •	Read announcements

How to Run
1.	Clone the repository
	git clone https://github.com/your-username/student-management-system.git
2.	Navigate the project
	cd student-management-system
3.	Install dependencies
	pip install flask werkzeug
4.	Run the application
	python app.py
5.	Open in browser
	http://127.0.0.1:5000/
